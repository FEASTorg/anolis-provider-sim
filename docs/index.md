# Anolis' Provider Simulator

You've reached the documentation for Anolis' Provider Simulator.
This project is a simulation tool designed to mimic the behavior of providers in a controlled virtual environment.
If you have gotten here, you are most likely looking to further develop the core anolis application or wish to develop your own provider.
In either case, the best place to start is [overview](overview.md) which gives a high-level introduction to the provider concept and usage.

## Local API docs

Run `doxygen docs/Doxyfile` from the repo root.
Generated HTML goes to `build/docs/doxygen/html/` and should not be committed.
